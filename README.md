# Панель управления учениками

Мини-приложение для менеджера онлайн-школы: список учеников, поиск, фильтрация, сортировка, карточка ученика с редактированием и форма создания нового ученика.

## Стек

- React
- TypeScript
- Vite
- React Router DOM
- Local mock API через `localStorage`

## Как запустить

```bash
npm install
npm run dev
```

Для production-сборки:

```bash
npm run build
```

## Что реализовано

- список учеников
- поиск по имени
- фильтр по статусу
- сортировка по дате регистрации
- открытие карточки ученика
- изменение статуса и комментария менеджера
- создание нового ученика с валидацией
- loading / error / empty states
- адаптивный интерфейс
- сохранение фильтров и открытой панели в URL query params

## Архитектурные решения

- `src/api/studentsApi.ts`:
  асинхронный слой данных, имитирует backend.
- `src/hooks/useStudents.ts`:
  управление загрузкой списка и синхронизацией после изменений.
- `src/components/*`:
  отдельные UI-части: тулбар, список, drawer, карточка ученика, форма создания, badge статуса.
- `localStorage`:
  созданные и измененные ученики сохраняются между перезагрузками страницы.

## UX-решения

- поиск дебаунсится и сохраняется в URL
- фильтры и сортировка сохраняются в query params, так что ссылкой можно поделиться
- карточка и создание открываются в боковой панели без потери контекста списка
- у загрузки, пустого результата и ошибок есть отдельные состояния
- кнопки и формы показывают `loading` во время асинхронных действий
- после создания ученик сразу появляется в списке и открывается его карточка
